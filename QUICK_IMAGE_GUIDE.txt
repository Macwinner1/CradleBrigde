🚀 FASTEST WAY TO ADD IMAGES (No Setup Needed!)
═══════════════════════════════════════════════════════════════════
1. Find an image online (or upload to imgur.com)
2. Copy the image URL
3. Login to admin dashboard
4. Create/Edit blog post
5. Paste URL in "Option 1: Paste Image URL" field
6. Click Save
✅ DONE! Image appears everywhere!

❌ Common Errors & Quick Fixes
═══════════════════════════════════════════════════════════════════
Error: Image doesn't show
Fix: Check URL is publicly accessible (open in new tab)

Error: "Firebase Storage not available"
Fix: Use Method 1 (Paste URL) instead

Error: Image preview fails
Fix: Make sure URL ends in .jpg, .png, or .gif

💡 Best Practices
═══════════════════════════════════════════════════════════════════
✅ Use HTTPS URLs (not HTTP)
✅ Test URL in browser first
✅ Recommended size: 1200x800px
✅ Keep file size under 500KB
✅ Use JPG for photos, PNG for graphics
